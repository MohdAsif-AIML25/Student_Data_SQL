# Student_Data_SQL
The Student Management System using SQL and Python is a mini project developed to manage student records efficiently using a database.
The project connects Python programming with an SQL database (SQLite) to perform basic database operations such as adding, viewing, updating, and deleting student details.

The main objective of this project is to understand how Python interacts with databases and how SQL queries are executed through a Python application in real time.
